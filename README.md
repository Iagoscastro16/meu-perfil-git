# Olá, sou Iago Castro

Formado em **Análise e Desenvolvimento de Sistemas** pelo SENAI Fortaleza. Trabalho como **Suporte Técnico** em uma empresa de ERP, onde cuido do banco de dados Firebird (2.5 e 3.0), configuro o Windows para receber o sistema, instalo o sistema tanto em servidores quanto em terminais, e treino usuários.

Também desenvolvo ferramentas internas que me ajudam no trabalho — o **NFCe Configurator** é uma delas. Atualmente estou trabalhando em um sistema para guardar tudo que é importante para pessoas como eu que trabalham no suporte técnico de uma empresa: resolução de bugs, informações fiscais, funcionamento do sistema e mensagens automáticas.

Estou desenvolvendo um site de jornal com **Django e Supabase** como projeto profissional. Meu principal objetivo na área de TI é me tornar um desenvolvedor **backend**.

📍 Fortaleza, Ceará — Brasil
